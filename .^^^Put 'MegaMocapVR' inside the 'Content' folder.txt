Do not put the entire extracted directory into your Unreal Engine project's 'Content' folder.

Only the "MegaMocapVR" folder should be put in 'Content' using Windows File Explorer (they cannot be dragged into Unreal Engine).

For more set up instructions, visit
https://github.com/Megasteakman/MegaMocapVR/wiki
